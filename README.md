# Car-Service-Database-Web-App

Project Description

This project represents a web application connected to a relational database designed for managing a car service activity.

The application allows the management of clients, cars, services, and appointments using a MySQL database and PHP backend.

The project was developed as part of a Database course assignment.

Technologies Used:
PHP, MySQL, HTML, CSS, JavaScript, XAMPP (local server)

Database Structure

The database includes the following main entities:
Clients, Cars, Services, Appointments.
Relationships between entities are implemented using foreign keys.

The SQL file required to create the database is included in this repository.


The application includes:

View records from database tables
Add new records
Update existing records
Delete records
Dropdown lists for foreign key selection
Complex queries (e.g., calculating total service cost)
Hidden primary and foreign keys from user interface

At least one page implements full CRUD operations.
